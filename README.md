# API Biblioteca

API REST para gerenciamento de livros, desenvolvida com Java e Spring Boot.

## Funcionalidades

- Cadastro de livros
- Busca por ID, título e ISBN
- Atualização e exclusão
- Paginação e ordenação
- Validação dos dados
- Tratamento global de exceções
- Controle de ISBN duplicado
- Autenticação com Spring Security
- Documentação com Swagger

## Tecnologias

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Spring Security
- MySQL
- MapStruct
- Bean Validation
- JUnit e Mockito
- Docker Compose
- Swagger/OpenAPI

## Como executar

1. Clone o repositório
2. Configure o arquivo `.env`
3. Execute o banco com Docker Compose
4. Inicie a aplicação
5. Acesse o Swagger
